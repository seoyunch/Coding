- Decomposition<br>
  브루트포스 문제여서 N부터 시작해서 0까지 포문을 돌려 해결

- WordStudy<br>
  기본적인 자바 문법을 사용해서 해결한 문제. 개발할떄에는 하나의 묶음 데이터를 테이블로 나누어 클래스나 레코드 형태로 처리했었기에 Map에 대한 개념이 부족한 상태로 문제를 해결하여 생각해내는데에 시간이 더 걸렸다.
  잊고있던 Map에대해 공부할 수 있었던 경험이었다. 앞으로도 부족한 자바 문법을 더 공부할 수 있을것 같다.

- Croatia<br>
